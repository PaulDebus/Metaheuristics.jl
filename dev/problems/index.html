Problems

Benchmark Test Problems for numerical optimization.

Metaheuristics.TestProblems.get_problem — Function
get_problem(problem)

Returns a 3-tuple with the objective function, the bounds and 100 Pareto solutions for multiobjective optimization problems or the optimal solutions for (box)constrained optimization problems.

Here, problem can be one of the following symbols:

  • :sphere
  • :discus
  • :rastrigin
  • :ZDT1
  • :ZDT2
  • :ZDT3
  • :ZDT4
  • :ZDT6

Example

julia> import Metaheuristics: TestProblems, optimize

julia> f, bounds, pareto_solutions = TestProblems.get_problem(:ZDT3);
